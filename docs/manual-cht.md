# 幻想空間 II 中文手冊要點

當年軟體世界代理的中文說明書《珍004-幻想空間Ⅱ》（骨灰集散地補完計畫掃描）要點索引。掃描原圖 41 頁不隨本 repo 散布（版權），本檔整理操作與遊玩要點。

## 遊戲操作（SCI0 EGA 文字冒險）

這是**打字式冒險遊戲**：你用鍵盤輸入英文指令來操作賴瑞，畫面回應為中文。新手先按 **F1** 看內建操作說明（已中文化）。

- **移動**：方向鍵或滑鼠點擊帶賴瑞走。
- **輸入指令**：在畫面下方輸入行打英文動作。引擎只認英文動詞，但回應是中文。常用指令：
  | 想做的事 | 輸入 |
  |---|---|
  | 觀察四周/物件 | `look`、`look at door`、`look at man` |
  | 跟人說話 | `talk to man`、`talk to woman` |
  | 拿/撿東西 | `get bottle`、`take money`、`pick up key` |
  | 開/關 | `open door`、`close window` |
  | 給/用東西 | `give ticket to man`、`use rope` |
  | 推/拉/按 | `push button`、`pull lever` |
  | 看道具欄 | `inventory` 或按 Tab |
  | 存檔/讀檔 | 選單 → 檔案，或 F5 存、F7 讀 |
- **選單**：按 **ESC** 或滑鼠點畫面頂端開選單（檔案 / 動作 / 速度 / 音效），可存讀檔、調速度、看道具。
- **存檔**：常存檔（F5）。卡關可讀檔（F7）重來。

## 右鍵顯示數字（debug 訊息）

若滑鼠右鍵/Shift-點擊時出現「43/153」之類數字，那是 SCI 引擎內建的**除錯座標顯示**（滑鼠 x/y 位置），非中文化造成，不影響遊玩。左鍵走路、打字下指令即可正常玩。

## 版權保護：電話號碼問答

開場顯示一位女子照片，要求輸入她在「小黑本」（電話簿）裡的電話號碼（`555-` 後四碼）。答錯兩次遊戲結束。

- **繁中版預設略過**：輸入任意四碼皆過關（原版對重玩者不友善）。中文提示畫面仍顯示。
- 想自己作答：遊戲內建 16 組號碼各對應一位女子照片，原版靠手冊電話簿頁對照（跑英文版可體驗原關卡）。

## 劇情背景

賴瑞·拉夫（Larry Laffer）在前作《鬼混蜥蜴樂園》找到真愛伊芙（Eve），但兩人未能長久。本作賴瑞參加電視相親節目、中了樂透，卻因一顆藏有微縮膠片的洋蔥被 KGB 探員與反派諾努基博士（Dr. Nonookee）追殺，展開橫跨郵輪、熱帶島嶼、火山口的荒謬冒險。

主要角色/地名譯名見 `../translation/GLOSSARY.md`。

## 音樂

原版內附 Roland MT-32 驅動。本中文化預設啟用 MT-32 模擬（音色遠勝 AdLib）。完整版附 ROM；patch 版玩家自備 MT-32 ROM 放入遊戲夾，於音效選項選 Roland MT-32。
