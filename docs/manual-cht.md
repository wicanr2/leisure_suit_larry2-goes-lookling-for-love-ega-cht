# 幻想空間 II 中文手冊要點

當年軟體世界代理的中文說明書《珍004-幻想空間Ⅱ》（骨灰集散地補完計畫掃描）要點索引。掃描原圖 41 頁不隨本 repo 散布（版權），本檔整理操作與遊玩要點。

## 遊戲操作（SCI0 EGA 文字冒險）

- **移動**：方向鍵或滑鼠點擊帶賴瑞走。
- **輸入指令**：直接打字下動作，如 `look`（看）、`talk to man`（跟人說話）、`open door`（開門）、`get object`（拿東西）、`inventory`（看道具）。中文化保留英文 parser 輸入（遊戲引擎只認英文動詞），畫面回應為中文。
- **選單列**：滑鼠移到畫面頂端出現選單（Action / Speed / Sound），可存讀檔、調速度與音效。
- **存檔**：常存檔。選單 Action → Save，或指令 `save`。

## 版權保護：電話號碼問答

開場顯示一位女子照片，要求輸入她在「小黑本」（電話簿）裡的電話號碼（`555-` 後四碼）。答錯兩次遊戲結束。

- 遊戲內建 16 組號碼，各對應一位女子照片。原版靠手冊的電話簿頁對照。
- **不想對照**：以 `SCI_CP_BYPASS=1` 環境變數啟動，任意四碼皆過關（見 README）。

## 劇情背景

賴瑞·拉夫（Larry Laffer）在前作《鬼混蜥蜴樂園》找到真愛伊芙（Eve），但兩人未能長久。本作賴瑞參加電視相親節目、中了樂透，卻因一顆藏有微縮膠片的洋蔥被 KGB 探員與反派諾努基博士（Dr. Nonookee）追殺，展開橫跨郵輪、熱帶島嶼、火山口的荒謬冒險。

主要角色/地名譯名見 `../translation/GLOSSARY.md`。

## 音樂

原版內附 Roland MT-32 驅動。本中文化預設啟用 MT-32 模擬（音色遠勝 AdLib）。完整版附 ROM；patch 版玩家自備 MT-32 ROM 放入遊戲夾，於音效選項選 Roland MT-32。
